# 卸载

本体位置   : `/Users/<User>/Library/Application Support/MhxyForMac`
启动器位置 : `/Applications/梦幻西游启动器.app`

E.g
```bash
rm -rf /Users/arylo/Library/Application\ Support/MhxyForMac
sudo rm -rf /Applications/梦幻西游启动器.app
```
